# Web Scraping Demo

This project demonstrates a simple Python web scraper using BeautifulSoup and a basic website to showcase the work.

## Files
- scrape_example.py: Python scraper script
- quotes.csv: generated dataset
- index.html: website landing page
- styles.css: styling
- script.js: small JavaScript enhancement

## How to run
1. Install dependencies:
   pip install requests beautifulsoup4
2. Run the scraper:
   python scrape_example.py
3. Open index.html in a browser to view the website.
